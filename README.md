# PaperQuantLab

Local-first multi-asset research workstation for explainable strategy experimentation, replay, journaling, and AI reasoning analysis.

## Philosophy

PaperQuantLab is NOT:
- a profit-maximizing auto trading bot
- an AI hedge fund
- a hype-driven autonomous trading system

It is a research-oriented simulation environment focused on:
- explainability
- replayability
- post-mortem analysis
- AI reasoning experiments
- human judgment training

## Key Features

### Outcome Intelligence
Automatically tracks the performance of every signal generated by the system.
- **Multi-interval tracking**: Monitors price delta at 5m, 15m, 1h, and 1d marks.
- **Strategy Insights**: Aggregates win rates and average returns per strategy.
- **Explainable Results**: Each outcome is tied back to the exact market regime and indicator snapshot at the time of the signal.
