# PaperQuantLab

Local-first multi-asset research workstation for explainable strategy experimentation, replay, journaling, and AI reasoning analysis.

## Philosophy

PaperQuantLab is NOT:
- a profit-maximizing auto trading bot
- an AI hedge fund
- a hype-driven autonomous trading system

It is a research-oriented simulation environment focused on:
- explainability
- replayability
- post-mortem analysis
- AI reasoning experiments
- human judgment training
